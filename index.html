React

I am exited to learn about React so I can build user interfaces or UI components for single page or mobile applications. It would be nice to build encapsulated components that manage their own state, then compose them to make complex UIs. I want to have the ability to develop new features in React without rewriting existing code from different platforms. React can also render on the server using Node and power mobile apps using React Native to make magic happen.
